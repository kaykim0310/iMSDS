## 📋 수정된 항목 구조

### 11. 독성에 관한 정보
- **가. 가능성이 높은 노출 경로에 관한 정보**
- **나. 건강 유해성 정보**
  - ○ 급성독성 (경구, 경피, 흡입(가스))
  - ○ 피부부식성 또는 자극성
  - ○ 심한 눈손상 또는 자극성
  - ○ 호흡기 과민성
  - ○ 피부 과민성
  - ○ 발암성 (산업안전보건법, 고용노동부고시, IARC, OSHA, ACGIH, NTP, EU CLP)
  - ○ 생식세포 변이원성
  - ○ 생식독성
  - ○ 특정 표적장기 독성 (1회 노출)
  - ○ 특정 표적장기 독성 (반복 노출)
  - ○ 흡인 유해성
  - ○ 기타 유해성 영향

### 12. 환경에 미치는 영향
- **가. 생태독성** (어류, 갑각류, 조류)
- **나. 잔류성 및 분해성** (잔류성, 분해성)
- **다. 생물농축성** (농축성, 생분해성)
- **라. 토양 이동성**
- **마. 기타 유해 영향**

### 15. 법적 규제 현황
- **가. 산업안전보건법에 의한 규제**
- **나. 화학물질관리법에 의한 규제**
- **다. 화학물질의 등록 및 평가 등에 관한 법률**
- **라. 위험물안전관리법에 의한 규제**
- **마. 폐기물관리법에 의한 규제**
- **바. 기타 국내 및 외국법에 의한 규제**
  - 국내규제
  - 기타 국내 규제
  - 국외규제
    - 미국관리정보 (OSHA 규정)
    - 미국관리정보 (CERCLA 규정)
    - 미국관리정보 (EPCRA 302 규정)
    - 미국관리정보 (EPCRA 304 규정)
    - 미국관리정보 (EPCRA 313 규정)
    - 로테르담협약물질
    - 스톡홀름협약물질
    - 몬트리올의정서물질

## 🔧 설치 방법

1. `kosha_api_extended.py` → **프로젝트 루트 폴더**에 복사
2. `pages/` 폴더의 파일들 → 기존 `pages/` 폴더에 덮어쓰기

## 🚀 사용 방법

1. Streamlit 앱 실행
2. **섹션 3**에서 구성성분의 **물질명**과 **CAS 번호** 입력
3. **섹션 11, 12, 15** 페이지에서 API 연동으로 데이터 조회 후 양식 작성

- 기존 데이터가 있는 경우 새 구조로 자동 변환됩니다

- API 조회 결과는 참고용이므로 반드시 검토 후 사용하세요
